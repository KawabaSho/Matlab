# Matlabで動くサンプルプログラムです
`sample_(number)_(name).m` は単体で動きます。`Demo_(number)_(name)` はフォルダ内に依存ファイルがまとまっています。フォルダ内の `main.m` を実行してください。

### 注意
`Demo_8` は、 任意の関数を自動微分する関数を生成し、自動微分を実行するプログラムですが、自動で `C/C++に変換` -> `OSSの自動微分ライブラリに埋め込み` -> `Matlabから呼び出すためのラッピング` -> `mexによる自動微分関数の生成` を行っているので、C/C++で書かれた外部ライブラリのCppADが必須です。CppAD は各自用意してください。
